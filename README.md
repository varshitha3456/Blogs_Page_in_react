# Blog App

A simple blog application built with React, featuring full CRUD functionality using the JSONPlaceholder API.

## Tech Stack
- React
- React Router DOM
- Axios
- Tailwind CSS

## Features
- View all posts
- Click a post to view full details
- Add, edit, and delete posts
- Navigation: Home, Posts, Contact (scrolls to footer)

## Folder Structure
```
src/
├── pages/
│   ├── Home.jsx
│   └── Posts.jsx
├── components/
│   ├── Navbar.jsx
│   ├── PostCard.jsx
│   └── PostDetail.jsx
└── App.js
```

## Run Locally
```bash
npm install
npm start
```
App runs at `http://localhost:3000`

## Vanilla JS Version
A plain HTML/CSS/JS version of the same project is available in the `vanilla-js/` folder. Open `vanilla-js/index.html` directly in a browser to view it.
